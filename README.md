## Project setup

Firefox 82.0.2, Chrome 86.0.4240.111 or later versions.<br />

Required:
  - Git
  - npm
<br />

### Running the project

  - `npm install` for installing the dependencies.<br />

  - `nodemon MailServer.js` for starting the nodemailer backend.<br />

  - `npm start` to run the project.<br />
